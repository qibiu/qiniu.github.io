你就是个神经病

Posted on 2019-05-05 Views:
Symbols count in article: 963 Reading time ≈ 1 mins.

你知道吗

以前对你无微不至好的不行，结果有一天，你特别郑重其事的跟我说“你压的我喘不过气”，“想要逃”，“感到窒息”，“需要空间”balabala都是你的原话，印象很深刻，那天我在找房子累的要死，说到房子本来住的好好的，为了你一句话，天天下班出去看房子。

从那之后其实挺心灰意冷的，然后如你所愿，对你没那么好了，结果你又说我不爱你了。

还有之前我觉得谈恋爱陪伴很重要。所以我下班之后的时间，基本上全部都用来陪你。你随叫，我随到。结果你觉得我腻，让我自己找点事情做，不要老陪着你，还说我个男生，连游戏也不打，简直不是男生。

好了，听你的，我玩游戏行了吧。然后你找我的时候，我在和朋友语音玩游戏，你又不开心。

还有世界杯期间，赌球，本来就是玩玩，花个一两百，给看球增添点气氛。别的女生或者说老婆，都是怕自己男朋友或老公深陷赌博出不来，家破人亡。你呢，一直叫我多赌点，一直说这是最后一次，我说不赌了，少赌点，你还生气，最后前前后后花了一两千，况且那时候手头也不充裕。我真的不知道说什么。

你冲我发脾气，我如果没忍住顶嘴了，那绝对是我的错，没毛病。但如果我没顶嘴，我憋在心里，那还是我的错。说是如果真的喜欢你，就不会对你有怨气。你总有你的理由，你总是对的，错的总是我。是的，我是假的喜欢你。

你莫名其妙发脾气的次数还少了吗，丰巢取件码你收到了，以为没收到，把我骂的狗血淋头，我顶嘴了吗？后来发现是冤枉我了，说自己很愧疚，下次会改，然后下次继续。

你的电动牙刷，没电了，你以为坏了，我只能通过你的描述，在网上找可能的原因，和解决办法。试了一两个没效果，就又冲我发脾气，我顶嘴了吗？

你的电脑连不上网，我也只能通过电话听你描述，找原因，给你找解决方法。也是试了两个，没效果，就开始冲我发脾气觉得我没用，我顶嘴了吗？还好最后有个方法起效果了，弄好了，你很开心说自己很聪明，是的你聪明，我没用。

这种屁事，我还能写一堆，你要是我，被我这样对待，你会怎么样？

这不，又想起一个。

你不小心打碎了你的玻璃烧水壶，心情不好，没错！冲我发脾气就对了宝贝！反正我对你来说，不就是个出气筒加钱包吗。真是人在家中坐，祸从天上来。然后给你买了个一模一样的，你才重新开心起来。没办法，你要是不开心了，第一个倒霉的就是我。

最后出气桶爆了，钱包瘪了，你也该走了，挥一挥衣袖不带走一片云彩。

感冒了
想二诺了
